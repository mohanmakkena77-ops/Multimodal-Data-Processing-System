Folder & File Purposes
--src/: All primary source code for processing, retrieval, and generation steps.
   --file_handlers/: Specialized scripts for extracting and processing data from each type of input.
   --rag/: Core RAG logic: retrieval from database and LLM-based answer generation.
   --database/: Scripts to connect and interact with the database or vector store.
   --utils/: Helper scripts, configurations, and common functions.
--ui/: (Optional) Scripts for your user interface (web, CLI, or app).
--samples/: Example files for demonstration and testing.
--tests/: Tests for verifying correct functionality.
--requirements.txt: All dependencies required for setup.
--README.md: Instructions, architecture, usage, and setup guide.
--.gitignore: Ensures temp, data, and secret files are not uploaded.
--setup.sh / install.bat: Installation or startup automation if needed.
--LICENSE: Compliance with open-source requirements.
